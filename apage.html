Zoomin

Zoomin is a non-profit organziation that helps bike enthusiast learn current bike trails in there area, along with current weather for the day and week. Intergrated with a Weather API and Trail API. Our Trail API shows users current trails in the city they live in. It can show current trail status, how busy is it and the popularity along with many other things. This was a group project in which we learned how to work with GitHub togther and our first time as a team. Our code consist of HTML, CSS and JavaScript. Along with the use of Web-APIs and Third party APIs.
